Volunteer Assistance Network (VAN)
    The Volunteer Assistance Network (VAN) aims to connect volunteers with individuals requiring help during disasters, such as floods. This Proof of Concept (POC) outlines essential modules and features for the initial development phase.

Modules: 
1. User Authentication:
   - Objective: Implement a secure user authentication system.
2. Location Services:
   - Objective: Enable location tracking for accurate volunteer and user positioning.
3. Volunteer Dashboard:
   - Objective: Provide a dashboard for volunteers to view assistance requests.
4. User Assistance Requests:
   - Objective: Allow users to request assistance during disasters.
5. Matching Algorithm:
   - Objective: Implement a basic algorithm to match volunteers with relevant assistance requests.
6. Communication Interface:
   - Objective: Facilitate communication between users and volunteers.

Usage: 
  In home page users can find the latest disaster news. Users can fill in their requests by clicking on the request section. After submitting they can check their request status in the website through their phone number. If they user wishes to nominate himself as volunteer they can register using the register section and they can start serving for the people's request nearby their location.
